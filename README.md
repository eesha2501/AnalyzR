## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/eesha2501/AnalyzR.git
   ```
2. Navigate to the project directory:
   ```sh
   cd AnalyzR
   ```
3. Install dependencies:
   ```sh
   npm install # or yarn install or pnpm install
   ```
